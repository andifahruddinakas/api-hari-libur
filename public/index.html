API Hari Libur Nasional Indonesia

Format Response

{
   "status" : "success",
   "code" : 200,
   "data" : {
      "date" : "2024-01-01",
      "description" : "Tahun Baru 2024"
   },
   "message": "Holidays found"
}

Akses API

  • /api untuk mendapatkan data hari libur tahun sekarang.
  • /api?year=2024 untuk mendapatkan data hari libur pada tahun 2024.
  • /api?month=1 untuk mendapatkan data hari libur di bulan 1.
  • /api?month=1&year=2024 untuk mendapatkan data hari libur di bulan 1 pada tahun 2024.

Sumber Data

API ini mengambil data dari tanggalan.com dengan melakukan scraping.
