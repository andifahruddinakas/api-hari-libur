API Endpoints

GET /api

Ambil data hari libur berdasarkan tahun dan bulan

# Semua hari libur 2026
curl "https://api-hari-libur.vercel.app/api?year=2026"
# Hari libur Januari 2026
curl "https://api-hari-libur.vercel.app/api?year=2026&month=1"

Query Parameters:

  • year - Tahun (default: tahun saat ini)
  • month - Bulan 1-12 (opsional)
POST /api/generate

Manual trigger untuk generate data tahun spesifik

# Generate data tahun 2027
curl -X POST "https://api-hari-libur.vercel.app/api/generate" \
-H "Content-Type: application/json" \
-d '{"{"}year": 2027{"}}'

Request Body:

{"year": 2027}
GET /api/scheduler/info

Cek status scheduler dan informasi otomasi

# Check scheduler status
curl "https://api-hari-libur.vercel.app/api/scheduler/info"
